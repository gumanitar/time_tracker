
# Time Tracker
- **Backend**: NestJS, Prisma, SQLite
- **Frontend**: React, Vite, TypeScript, React Router, TanStack React Query, Material-UI


## Installation

### Backend Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run Prisma migrations:
   ```bash
   npx prisma migrate deploy
   ```

4. Start
   ```bash
   npm start
   ```

### Frontend Setup
In a new terminal:
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3.  Start 

```bash
npm run dev
```

The frontend development server will run on `http://localhost:5173`
